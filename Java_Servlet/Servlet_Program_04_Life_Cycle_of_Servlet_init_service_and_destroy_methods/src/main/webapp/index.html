Servlet Life Cycle

1. Loading and Instantiation

-- First tomcat server started
-- java files (servlet file, than extend HttpServlet and ather java file) will compiled and class file will be created
-- Then class file will be loaded in the memory
-- and servlet object is created (servlet class file object)

2. Initialization

-- call init() method.
-- servlet object will be Initializing by invoking init() method

3. Request Handling

-- call service() method.
-- it will hendl Client Request.
-- if Client Request is get method than service() method call doGet() method.
-- if Client Request is post method than service() method call doPost() method.
-- Client Request send it is created a new thread and then this thread hendl Request (call service() method and this method resolve client request).
-- Then Client send another request this created a new thread this thread hendl client request... these proses continuously running as per client request
-- it is multi threading precess

4. Destroy

-- server stopped and close database connection (if connected!) then call destroy() method.
-- destroy() method delete servlet object

Note: step 1, 2 and 4 it runs only one time. init() and destroy() method call one time, but step 3 run multiple time, service() method call many times
